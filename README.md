Best of the Left Podcast<br>
=============================

<a href="www.kodi.tv">KODI</a> / XBMC / Firestick audio addon for <a href="http://www.bestoftheleft.com">Best of the Left podcast</a>.<br>

<a href="http://www.bestoftheleft.com">Best of the Left podcast</a>

^^^ To install this addon, either use the <a href="https://www.tvaddons.co/github-browser-kodi/">Kodi Github installer</a> addon or save the .zip file downloaded from the 'clone or download' button to somewhere the Kodi can access (e.g. network drive or USB stick). Then on the Kodi, go to addons > install from zip file.<br>
